项目名：品优购项目

了解：
  网站制作流程：
    客户沟通 -> 签订合同 -> 预付订金 -> 审核需求 ->
    前端页面设计、后台功能开发 -> 测试 -> 上线 -> 维护

  收缩引擎优化：
    SEO优化：
        Search Engine Optimization，一种利用搜索引擎的规则来提高网站在搜索结果中的排名
    三大标签/属性，TDK：
        title         建议写作：网站名(产品名)-简要介绍
        description   网站描述
        keywords      关键字，建议6-8个词组成，逗号分隔


项目规划：
  制作：首页、列表页、注册页
  准备：
      项目文件夹        Shopping
      样式类图片文件夹   images
      样式文件夹        css
      产品类图片文件夹   upload
      字体类文件夹      fonts
      脚本文件夹        js
  模块化开发：
      首页文件           index.html
      初始化样式文件      base.css      参考京东的样式文件
      公共样式           common.css    常用于不同页面中不变的模块，如顶栏、底栏
  
  设置title中的网页图标，一般写作 favicon.ico
      制作图标、图标放在项目根目录下、HTML页面引入图标

    1，制作：
      把表示图标的.png图片转换为.ico格式的图片（利用第三方网站：www.bitbug.net）
    2，转换后置于项目根目录下
    3，HTML页面引入图标
        <link rel="shortcut icon" href="favicon.ico" type="image/x-icon" />

注意：
    应该先制作不同页面公有的模块（固定不变的），如顶栏、底栏

一、制作公共模块
      1，快捷导航栏：
          通栏、版心居中、一个左浮动盒子、一个优浮动盒子
      2，header头部：
          LOGO区域、搜索区域、快捷导航、购物车区域
      3，nav导航栏
          左侧商品分类、右侧导航单元
      4，底部通栏
          版心居中，包含服务模块、帮助模块、版权模块
      
二、首页专有的模块
      1，有main主体模块，对应的样式放在 index.css 文件中
          其中，中间为轮播图、右侧为新闻模块
      2，推荐模块，推荐展示产品信息
