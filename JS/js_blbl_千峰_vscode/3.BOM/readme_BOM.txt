BOM：
  浏览器对象模型，Browser Object Model

关于 window 对象：
  1，打开页面时，就有一个window对象
  2，windows下包含所有的全局属性和方法（包含自定义的）
  3，BOM的顶级对象，也是前端的顶级对象
  4，前端JS最大可操作范围就是 window
  5，所有和BOM相关的API都以：window.xxx 表示，所以全局属性和方法的完整写法为 window.xxx
  6，全局的属性和方法的写法都可以省略 window. 部分

用于操作浏览器：
  地址栏
  滚动条
  前进后退刷新
  等等
